אפיון הפרויקט

עמוד מתומצת

RingLore הוא אתר אינטרנט לימודי-תוכני בנושא עולמו של J.R.R. Tolkien ו-The Lord of the Rings. האתר מציג מידע על הדמויות המרכזיות, יצורים קסומים, הסופר, סיפור העולם ומפה אינטראקטיבית של Middle-earth. בנוסף, האתר כולל מערכת הרשמה, התחברות וניהול משתמשים המבוססת על מסד נתונים מקומי.

תחום וסוג

  • תחום: אתר תוכן אינטראקטיבי עם רכיב ניהול נתונים.
  • סוג: Web Application בטכנולוגיית ASP.NET Web Forms.
  • נושא: עולם שר הטבעות, דמויות, יצורים ומיקומים.

מטרת הפרויקט

  • ליצור אתר מאורגן שמרכז מידע למעריצים וללומדים.
  • להדגים עבודה עם טפסים, Session, מסד נתונים ופעולות CRUD.
  • לאפשר ניווט פשוט בין עמודי תוכן לבין עמודי משתמשים.

קהל יעד

  • משתמשים שרוצים להכיר את עולם Middle-earth.
  • תלמידים או קוראים המחפשים תקצירי מידע מסודרים.
  • מנהל אתר שצריך לצפות, לסנן, לעדכן ולמחוק משתמשים.

שימושים עיקריים

  • קריאת מידע על דמויות, יצורים, הסופר ועלילת LOTR.
  • חקירת מיקומים דרך מפה אינטראקטיבית.
  • הרשמה והתחברות למערכת.
  • ניהול רשימת משתמשים דרך טבלה.

Code Behind

שמירת משתמש חדש - Signup.aspx.cs

protected void btnSend_Click(object sender, EventArgs e)
{
    User user = new User();
    user.fName = Request.Form["fName"];
    user.lName = Request.Form["lName"];
    user.username = Request.Form["username"];
    user.pass = Request.Form["pass"];
    user.city = Request.Form["city"];
    user.phone = Request.Form["phone"];
    user.email = Request.Form["email"];
    user.address = Request.Form["address"];
    user.gender = Request.Form["gender"];

    DateTime bd;
    if (DateTime.TryParse(Request.Form["birthday"], out bd))
        user.birthday = bd;

    Helper.Insert(user);
    Response.Redirect("Home.aspx");
}
הקוד אוסף את שדות הטופס, יוצר אובייקט User, ממיר תאריך לידה אם הוזן נכון, ושולח את האובייקט למחלקת העזר שמבצעת את פעולת ההכנסה למסד הנתונים.

פעולת Insert עם פרמטרים - Helper.cs

using (SqlCommand cmd = new SqlCommand(
  "INSERT INTO tblUsers (FirstName,LastName,Username,Password,Town,BirthDate,gender,Email,tel,address) " +
  "VALUES (@f,@l,@u,@p,@c,@b,@g,@e,@t,@a)", con))
{
    cmd.Parameters.AddWithValue("@f", user.fName ?? (object)DBNull.Value);
    cmd.Parameters.AddWithValue("@u", user.username ?? (object)DBNull.Value);
    cmd.Parameters.AddWithValue("@p", user.pass ?? (object)DBNull.Value);
    con.Open();
    cmd.ExecuteNonQuery();
}

סינון משתמשים

public string BuildSQLStr(string filter)
{
    if (filter.Length == 0)
        return "SELECT * FROM tblUsers";

    string SQLStr =
      $"SELECT * FROM tblUsers WHERE " +
      $"FirstName LIKE '%{filter}%' OR " +
      $"LastName LIKE '%{filter}%' OR " +
      $"Username LIKE '%{filter}%'";

    return SQLStr;
}
בהתחברות נעשה שימוש בפרמטרים כדי לצמצם סיכון להזרקת SQL. בפעולת הסינון בדף Users נבנית שאילתה דינמית; שיפור עתידי מומלץ הוא להפוך גם אותה לשאילתה עם פרמטרים.

מה למדתי מהפרויקט

במהלך בניית RingLore למדתי לחבר בין עיצוב אתר, טפסים, קוד שרת ומסד נתונים. הפרויקט דרש עבודה עם Master Page, דפי ASPX, קבצי Code Behind, שאילתות SQL, Session וניהול נתונים. בנוסף למדתי לארגן תוכן רב בצורה נוחה למשתמש ולשלב תמונות וניווט ברור.

מה ניתן לשפר

  • הצפנת סיסמאות במקום שמירה כטקסט רגיל.
  • הוספת ולידציה מלאה בצד לקוח ובצד שרת.
  • שימוש בפרמטרים בכל השאילתות, כולל סינון.
  • הפרדה טובה יותר בין עיצוב CSS לבין קוד HTML.

אפשרויות עתידיות

  • מערכת הרשאות מלאה למנהל ולמשתמש רגיל.
  • חיפוש מתקדם בדמויות, יצורים ומיקומים.
  • הוספת שאלון ידע או משחק טריוויה בנושא LOTR.
  • מעבר למבנה מודרני יותר כמו ASP.NET MVC או ASP.NET Core.
מסקנה: RingLore משלב אתר תוכן עם יכולות מערכת בסיסיות, ולכן הוא מדגים גם בניית ממשק משתמש וגם עבודה מעשית מול מסד נתונים.
